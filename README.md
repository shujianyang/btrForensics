#btrForensics
Forensic Analysis Tool for Btrfs File System.

###Platform:
Linux

###Prerequisite:
Install the Sleuth Kit library --> [Link](https://github.com/sleuthkit/sleuthkit.git)

###Input File:
Raw image of a btrfs partition.
